# ORGANIZING Zom-Ga FILES
### In this file, I'll be explaining what each file has where it belonds for any other developer checking this content or for future development by me:
note: You can skip this part if you're uninterested, but since this fill is like a manual for how you turn on the game!

> ZG.py
- This file is the main file where the game should be started from
- it has most of the other files connected to it

> Settings.py
- This is the file where the main settings of the game and the main definitions exist

> Scenes.py
- this file is where the scenarios definitions exists

> Randomizer.py
- this is just basically like the dice for the events to take holds with the weights of each events

> Lists.py
- this is the file where the possibilities of each event occurs
- the events are separated into **_FIVE_** Categories:
    1. Home Events
    2. Roof Events
    3. Outside Events
    4. Legend Events
    5. Ultra Legend Events

> Music.py (does not exist yet)
- this file where the main files of sound effects and music are located
- they are connected to Sound folder, where the main sound effects, music, and everything else related to sound settings are
